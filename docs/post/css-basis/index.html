Selector

  • a simple selector

    • universal selector *
    • type selector p
    • class selector .class
    • attribute selector :[href]
    • id selector #id
    • pseudo class :hover

  • a sequence of simples selectors

    • always starts with type selector or universal selector
    • pseudo elements must be the last selector if any

  • a selector

:not()

  • :not() takes simple selectors excluding :not() itself as an argument.
  • nested negation :not(:not()) is invalid

Selector Specifity

The selector specifiy is defined as a, b, c, where

  • ID selectors $\rarr$ a
  • class/attribute/pseudo-classes selectors $\rarr$ b
  • type/pseudo elements $\rarr$ c
  • ignore the universal selector

Efficient Selector

  • avoid writing universal selector
  • ID selector is unique, so there is no need to add other parts
  • recommend inheritence
  • recommend child combinator rather descendant combinator
